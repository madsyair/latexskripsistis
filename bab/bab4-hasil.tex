\section{Contoh Grafik R sebagai LaTeX (Font Serasi)}
\emph{Cara menulis.} Agar \textbf{font di dalam grafik sama persis dengan font naskah}
(Times New Roman/TeX Gyre Termes), buat grafik di R dengan paket \texttt{tikzDevice}:
keluarannya berupa berkas \texttt{.tex} (kode Ti\emph{k}Z) yang di-\texttt{input} ke
naskah dan dikompilasi XeLaTeX bersama teks, bukan gambar raster. Jalankan kode R
berikut sekali (di luar Overleaf, mis.\ di RStudio), lalu unggah berkas \texttt{.tex}
hasilnya ke folder \texttt{img/}.

\begin{lstlisting}
library(tikzDevice)
options(tikzDefaultEngine = "xetex")
options(tikzXelatexPackages = c("\\usepackage{tikz}",
        "\\usepackage{fontspec}", "\\setmainfont{TeX Gyre Termes}"))
tikz("img/plot-r.tex", width = 4.6, height = 3.0, standAlone = FALSE)
op <- par(mar = c(4, 4, 1, 1), family = "serif")   # 'serif' -> font naskah
x <- 1:6; y <- c(11.2, 10.8, 9.9, 9.1, 8.4, 7.9)
plot(x, y, type = "b", pch = 19,
     xlab = "Tahun ke-", ylab = "Tingkat kemiskinan (\\%)")
par(op); dev.off()
\end{lstlisting}

Gambar~\ref{gbr:rtikz} menyisipkan berkas tersebut; bandingkan font label sumbu dan
angkanya dengan teks paragraf ini --- keduanya identik. Karena grafik berupa vektor, ia
tidak pecah saat diperbesar; ukuran diatur lewat \texttt{width}/\texttt{height} (inci)
pada \texttt{tikz()}.

\section{Contoh Grafik dengan Simbol Matematika (Regresi)}
\emph{Cara menulis.} Karena grafik tikz berupa kode LaTeX, \textbf{anotasi di dalam
grafik dapat memuat simbol matematika} (mis.\ $\beta_0$, $\beta_1$, $\hat{y}$, $R^2$)
yang identik dengan simbol pada teks. Tulis anotasi sebagai matematika LaTeX di dalam
\texttt{\$...\$} pada argumen \texttt{labels}/\texttt{xlab}/\texttt{ylab}. Jalankan kode
R berikut, lalu unggah \texttt{img/plot-regresi.tex} hasilnya.

\begin{lstlisting}
library(tikzDevice)
options(tikzDefaultEngine = "xetex")
options(tikzXelatexPackages = c("\\usepackage{tikz}",
        "\\usepackage{amsmath}", "\\usepackage{fontspec}",
        "\\setmainfont{TeX Gyre Termes}"))
set.seed(7); x <- runif(40, 1, 10); y <- 2.3 + 1.8*x + rnorm(40, sd = 2.2)
fit <- lm(y ~ x); b <- coef(fit); r2 <- summary(fit)$r.squared
tikz("img/plot-regresi.tex", width = 5.0, height = 3.4, standAlone = FALSE)
op <- par(mar = c(4, 4, 1, 1), family = "serif")
plot(x, y, pch = 19, col = "grey30",
     xlab = "Pengeluaran per kapita ($x$, juta Rp)",
     ylab = "Skor kesejahteraan ($y$)")
abline(fit, col = "red", lwd = 2); ya <- max(y)
text(2.2, ya - 0.5, adj = 0, labels = "$\\hat{y} = \\beta_0 + \\beta_1 x$")
text(2.2, ya - 3.0, adj = 0,
     labels = sprintf("$\\hat{y} = %.2f + %.2f\\,x$", b[1], b[2]))
text(2.2, ya - 5.5, adj = 0, labels = sprintf("$R^2 = %.3f$", r2))
par(op); dev.off()
\end{lstlisting}

Gambar~\ref{gbr:reg} menyajikan garis regresi beserta persamaannya dalam bentuk simbol
($\hat{y} = \beta_0 + \beta_1 x$) dan bentuk terisi (koefisien taksiran). Simbol
$\beta_0$, $\beta_1$, $\hat{y}$, dan $R^2$ pada grafik dan pada teks ini dirender mesin
yang sama sehingga seragam; pendekatan ini berlaku untuk simbol apa pun (mis.\ $\sum$,
$\bar{x}$, $\sigma^2$, $\hat{\beta}$).

\section{Contoh Penomoran dan Rujukan Silang}
\emph{Cara menulis.} Penomoran bab/subbab, persamaan, tabel, dan gambar dilakukan
\textbf{otomatis} oleh LaTeX; cukup beri \texttt{\textbackslash label\{...\}} lalu rujuk
dengan \texttt{\textbackslash ref\{...\}} (persamaan dengan \texttt{\textbackslash
eqref\{...\}}). Nomor akan menyesuaikan sendiri bila urutan berubah, sehingga tidak
perlu menomori manual.

Sebagai contoh \textbf{persamaan bernomor}, koefisien Gini dirumuskan sebagai
\begin{equation}\label{eq:gini}
G = \frac{\sum_{i=1}^{n}\sum_{j=1}^{n} |x_i - x_j|}{2 n^2 \bar{x}}
\end{equation}
sehingga ketimpangan pada persamaan~\eqref{eq:gini} dihitung dari data pengeluaran.
Rujukan antarobjek bersifat dinamis: Tabel~\ref{tab:miskin} dan
Gambar~\ref{gbr:bar}/\ref{gbr:rtikz} otomatis memuat kata jenis beserta nomornya.

\textbf{Daftar bernomor} ditulis dengan lingkungan \texttt{enumerate}:
\begin{enumerate}
  \item Tahap pertama: praproses dan pembersihan data.
  \item Tahap kedua: pemodelan dan validasi.
  \begin{enumerate}
    \item subbutir dengan huruf;
    \item subbutir berikutnya.
  \end{enumerate}
  \item Tahap ketiga: evaluasi dan interpretasi.
\end{enumerate}
